## 🧠 What Is This Course?

The **5-Day AI Agents Intensive** by Kaggle and Google is a hands-on sprint through the new era of AI-assisted software development — where developers stop being pure coders and start becoming **system orchestrators**: designing the evaluation, constraint, and context harnesses that allow AI agents to operate safely and effectively.

Each day combines:
- 🎙️ A podcast / whitepaper deep-dive
- 🛠️ Hands-on codelabs
- 🚀 A real deployed artifact
